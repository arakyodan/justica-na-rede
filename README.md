# Projeto Justiça na Rede - Combate ao Cyberbullying  #

## Sobre o Projeto##
Intervenção social desenvolvida para a disciplina de Direito [Nome da Disciplina] na **Universidade Potiguar (UNP)**. 
O objetivo é fornecer uma ferramenta digital interativa para auxiliar vítimas de intimidação sistemática no ambiente virtual.

## Tecnologias Utilizadas##
- HTML5 / CSS3
- JavaScript (Lógica do Simulador de Crimes)
- GitHub Pages (Hospedagem)

##Base Jurídica##
Este projeto fundamenta-se nas seguintes normas:
- **Lei 14.811/2024:** Tipificação do crime de Cyberbullying (Art. 146-A CP).
- **Código Civil:** Responsabilidade civil e reparação de danos (Arts. 186 e 927).
